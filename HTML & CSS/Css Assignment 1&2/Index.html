Start Making a HTML File

After selecting a HTML editor, now you can start making a HTML file:-


Below are the steps of tags thet you need to make a HTML page.

  1. Start by declaring what type of file you creating and this is done using this tag !DOCTYPE html,and include the opening tags.
  2. Now include html&/html on the next line.
  3. Open the body tags inside the html tags. body&/body
  4. Open a heading for your page as h1 & /h1
  5. Put a title "Put a title between the tags" /title tags in the header tag.

    6. Now save the page as index.html and run it.
